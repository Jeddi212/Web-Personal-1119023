    Saya Jedediah Fanuel, biasa dipanggil Jeddi, atau Jed, atau Je, atau Jedai, atau Jedong, atau Jedut, atau Jedot, dll. Saya lahir di kota Bandung pada tanggal 28 April 2001. Saya memiliki seorang kaka perempuan bernama Jeconiah Fanuel. Ia sekarang berumur 19 tahun setahun lebih tua dari saya.

    Kini saya menjalani studi di Institut Teknologi Harapan Bangsa. Di sini rasanya sangat seru dan bersahabat. Seru dan tidak terlalu serius dan berkompetisi, lebih kepada arah kolaborasi. Walaupun di sini kurang aktivitas dan peminatnya dalam menjalani UKM tetapi saat pembelajaran, para mahasiswa belajar penuh dengan semangat dan sangat antusias.

    Saya memiliki cita-cita menjadi seorang programmer. Meskipun begitu, dulu saya bercita-cita menjadi seorang pemadam kebakaran XD. Tetapi saya rasa membuat program tidak semudah dan juga tidak sesulit yang saya pikirkan, initinya susah-susah gampang. Tapi saya benci matematika sih. Entah kenapa kalau suatu pelajaran saya tidak tahu untuk apa manfaatnya bagi saya, saya jadi sulit untuk menyerapnya ke dalam otak saya. Entahlah apa manfaat dari Kalkulus bagi kehidupan sehari-hari saya. Mungkin biar ada kerjaan di kampus.
